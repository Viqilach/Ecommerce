# 🕯️ LUMA — Aromaterapia Artesanal

## Objetivo 🎯

Diseñar un ecommerce de temática libre que permita visualizar productos, filtrarlos por categoría, ordenarlos por precio y simular una sesión de usuario con carrito de compra.

El ecommerce funciona tanto en móvil como en escritorio.

---

## Funcionalidades ⚙️

- 🛒 Gestión del carrito: añadir y eliminar productos
- 💰 Cálculo del total con coste de envío
- 🗂️ Filtrar productos por categoría
- 🔃 Ordenar de menor a mayor precio
- 🔐 Simulación de sesión de usuario 
- 🌿 Modal "Nosotras" con la historia de la marca
- 📦 Dropdown de colecciones en el menú de navegación

---

## Tecnologías 💻

- HTML
- CSS
- JavaScript
- Git
- GitHub

---

## ¿Cómo ejecutarlo? 🚨

1. Clonar el repositorio.
2. Abrir el archivo `index.html` en el navegador.

El ecommerce se cargará automáticamente.
